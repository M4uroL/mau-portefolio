SGBD para Controle de Propinas

Desenvolvi um SGBD relacional utilizando MySQL para o controle eficiente de propinas em uma instituição de ensino. O sistema permite: Registo de estudantes e cursos Acompanhamento de pagamentos Geração de relatórios financeiros Controle de mensalidades em atraso Esse projeto reforçou meu domínio em modelagem de banco de dados, relações entre tabelas e uso de comandos SQL para automação de tarefas administrativas.

Sistema Bancário Simples

Desenvolvi um sistema bancário simples em Java, com funcionalidades básicas de gestão de contas, com funcionalidades implementadas: Criação de contas bancárias, Depósitos e saques com verificação de saldo, Transferências entre contas, Consulta de extrato, Armazenamento de dados em memória, Este projeto reforçou conceitos de Programação Orientada a Objetos, como herança, encapsulamento e polimorfismo, além de lógica de programação e boas práticas na estruturação de código.
